# Contribuindo

Obrigado pelo interesse em contribuir! Toda contribuição é bem-vinda.

## Como Contribuir

### Reportando Bugs

1. Verifique se o bug já não foi reportado nas [Issues](https://github.com/LuisMIguelFurlanettoSousa/myPortfolio/issues)
2. Abra uma nova issue usando o template de **Bug Report**
3. Inclua o máximo de detalhes possível: passos para reproduzir, comportamento esperado vs. atual, screenshots

### Sugerindo Melhorias

1. Abra uma issue usando o template de **Feature Request**
2. Descreva claramente a melhoria e o problema que ela resolve

### Enviando Pull Requests

1. Faça um fork do repositório
2. Crie uma branch a partir da `master`: `git checkout -b feat/minha-feature`
3. Faça suas alterações seguindo os padrões do projeto
4. Commit com mensagens semânticas em português (ex: `feat: adicionar animação no hero`)
5. Push para seu fork: `git push origin feat/minha-feature`
6. Abra um Pull Request descrevendo suas alterações

## Padrões do Projeto

- **Commits semânticos** em português (`feat:`, `fix:`, `refactor:`, `docs:`, `chore:`)
- **TypeScript** com tipagem estrita
- **Componentes React** seguindo princípios SOLID e componentização adequada
- **Tailwind CSS** para estilização
- Rode `npm run lint` antes de enviar seu PR

## Ambiente de Desenvolvimento

```bash
git clone https://github.com/LuisMIguelFurlanettoSousa/myPortfolio.git
cd myPortfolio
npm install
npm run dev
```
